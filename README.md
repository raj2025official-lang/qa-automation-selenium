# 🚀 QA Automation Project (Selenium + Pytest)

## 🎯 Purpose

This project demonstrates end-to-end automation testing of an e-commerce website using Selenium and Pytest with Page Object Model (POM) design.

---

## 📌 Project Overview

This project automates testing for a sample website (SauceDemo), covering login, product selection, cart, and checkout flow.

---

## 🚀 Features

* Page Object Model (POM) design pattern
* Automated login functionality
* Add single and multiple products to cart
* Cart validation
* Checkout process automation
* Negative test cases

---

## 🧪 Test Cases Included

* Google Homepage Test
* Python.org Search Test
* SauceDemo Purchase Test
* Login Negative Test
* Multiple Products Test
* Price Validation Test

---

## 🛠 Tech Stack

* Python
* Selenium WebDriver
* Pytest

---

## 📂 Project Structure

## ▶️ How to Run

```bash
pip install -r requirements.txt
pytest -v
```
